# 모듈형 수직 농장 프레임 설계 가이드 (KF-MODULAR-FRAME-600)
**Project:** Modular Plant-terior System (DIY Flat-pack)
**Unit Type:** 600x600 Atomic Module (Base Unit)
**Version:** 1.0 (Initial Design)

---

## 🚀 설계 철학: "IKEA for Smart Farming"
1. **표준화**: 600mm x 600mm 정사각 베이스를 통해 공간 효율 및 조립성 극대화.
2. **모듈성**: 가로/세로 무한 확장이 가능한 연결 인터페이스 설계.
3. **사용자 맞춤형**: 사용자 환경(카페, 식당, 가정)에 맞게 외장재(Wood, Glass, PC)를 선택 교체.
4. **물류 효율**: 모든 부품을 분해하여 납적한 패키지(Flat-pack)로 배송 가능한 구조.
5. **자원 최적화**: 모듈 결합 시 기둥과 프레임을 공유하여 자재 낭비 및 비용 절감.

---

## 🏗️ 1. 구조 설계 전략 (Vertical Stacking)
가로 확장뿐만 아니라 **수직 적층(Vertical Stacking)**을 통해 공간 효율을 극대화합니다.

- **스탠다드 2단형 (Standard 2-Tier)**: 
    - 바텀 허브(300mm) + 재배 유닛(600mm) x 2 = **총 높이 1500mm**.
    - 일반적인 카페 카운터 높이 및 성인 시선 높이에 최적화된 표준형.
- **프리미엄 3단형 (Premium 3-Tier)**: 
    - 바텀 허브(300mm) + 재배 유닛(600mm) x 3 = **총 높이 2100mm**.
    - 천장이 높은 상업 공간에서 압도적인 쇼케이스 효과를 주는 하이엔드형.

| 구분 | 부품명 | 상세 사양 | 수량 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Profile** | **알루미늄 프로필** | - 2020 또는 2040 T-Slot<br>- 아노다이징 블랙/실버<br>- 길이: 600mm | 8 | 가로/세로 빔 |
| **Bracket** | **3-Way 코너 브라켓** | - 알루미늄 다이캐스팅<br>- 조립용 볼트/너트 포함 | 8 | 프레임 결속 |
| **Leveler** | **조절좌 (Foot)** | - M8~M10 나사산 가공<br>- 높낮이 조절 및 진동 방지 | 4 | 최하단 모듈용 |
| **Snap-fit** | **모듈간 커넥터** | - 상하/좌우 모듈 결합용<br>- 퀵 릴리즈 타입 스냅핏 | 4 | 모듈 확장용 |
