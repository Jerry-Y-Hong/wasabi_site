# 스마트팜 양액공급기 개발 기능 명세서 (Functional Specification)
**Project Code:** KF-NUTRI-2026 (가칭)
**Target Grade:** 중~고급형 스마트팜 복합환경제어 연동 표준 모델
**Version:** 1.0

## 1. 개요 (General Overview)
본 문서는 국내 시설원예 스마트팜 환경에 최적화된 양액공급 시스템의 하드웨어 및 소프트웨어 요구사항을 정의한다. 기존 시장 선도 제품(신한에이텍, 우성하이텍 등)의 장점을 벤치마킹하고, 최신 스마트팜 국가 표준(KS X 326x)을 준수하는 것을 목표로 한다.

## 3. 제어 기능 (Control Functions)

### 3.1 양액 제어 로직
1.  **EC/pH 기반 피드백 제어:**
    *   설정된 목표 EC/pH 값에 도달하도록 A, B, C 밸브 개폐율을 PID 제어.
    *   원수 EC 보정 기능 포함 (원수 수질 변화 자동 대응).
2.  **비례 제어 (Ratio Control):**
    *   유량 변화에 따른 양액 흡입량 비례 제어.

### 3.2 관수 스케줄링 (Irrigation Scheduling)
1.  **시간 제어 (Time Based):** 시작 시간, 종료 시간, 관수 간격 설정.
2.  **일사량 제어 (Solar Radiation / Joule Integrator):**
    *   누적 광량(J/cm²) 도달 시 자동 관수 (예: 매 100J 마다 1회).
    *   흐린 날/맑은 날 자동 구분 관수량 조절.
3.  **외부 트리거 (External Trigger):**
    *   토양 수분 센서 또는 배지 중량 센서 연동 관수.

### 3.3 안전 및 경보 기능 (Safety & Alarms)
*   **고농도/저농도 경보:** 설정 범위 이탈 시 즉시 급액 중단 및 알람.
*   **유량 이상 감지:** 펌프 공회전 방지 및 배관 누수/막힘 감지.
*   **센서 오류 감지:** EC/pH 센서 단선 또는 비정상 수치 지속 시 비상 운전 모드 전환.

## 4. 사용자 인터페이스 (HMI & Connectivity)

### 4.1 로컬 제어 (HMI)
*   **디스플레이:** 10.1인치 정전식 터치스크린 (Industrial Panel PC).
*   **OS/Software:** Windows IoT 또는 Linux 기반, 국문/영문 지원 GUI.
*   **UI 구성:** 실시간 모니터링 대시보드, 설정 마법사, 트렌드 그래프(과거 데이터 조회).

### 4.2 원격 제어 및 데이터 (Remote & Data)
*   **클라우드 연동:** LTE/Wi-Fi 모듈 탑재, 스마트폰 앱을 통한 원격 모니터링/제어.
*   **데이터 로깅:** 모든 관수 이력(일시, 공급량, 평균 EC/pH 등) 1년 이상 로컬 저장 및 서버 전송.
*   **표준 프로토콜:** KS X 3267 (스마트 온실 제어 인터페이스) 지원 권장.
